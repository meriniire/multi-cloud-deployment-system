# 🌟 Multi-Cloud Container Deployment System

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![Streamlit](https://img.shields.io/badge/Streamlit-1.28.0-red.svg)](https://streamlit.io/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

A professional multi-cloud container deployment platform that enables seamless deployment, monitoring, and management of containerized applications across AWS, Azure, and Google Cloud Platform (GCP).

## 🚀 Features

- **Multi-Cloud Support**: Deploy applications to AWS, Azure, and GCP from a single interface
- **User Authentication**: Secure login and registration system with role-based access (Student, Teacher, DevOps, Admin)
- **Container Orchestration**: Deploy and scale containerized applications with configurable replicas
- **Real-time Monitoring**: Track CPU usage, memory utilization, and deployment metrics across all clouds
- **Workload Management**: Scale, restart, and manage deployed containers
- **Infrastructure as Code**: Built-in support for Terraform, Pulumi, and CloudFormation templates
- **Cost Optimization**: Track spending and get intelligent cost-saving recommendations
- **Beautiful UI**: Navy blue, gold, and cream themed responsive interface

## 📋 Prerequisites
